# Previsão de Aprovação de Empréstimos

Este projeto tem como objetivo prever a variável binária Loan_Status (Aprovado ou Rejeitado). Trata-se de um problema clássico de classificação supervisionada, ideal para testar e comparar algoritmos como Random Forest, Regressão Logística, SVM, entre outros.

A análise envolve desde a preparação dos dados até à avaliação detalhada das métricas dos modelos, permitindo compreender quais fatores influenciam mais a aprovação do empréstimo.

## Dataset

O dataset utilizado é o Realistic Loan Approval Dataset | US & Canada, que contém atributos relevantes para o processo real de decisão de crédito, tais como:

cibil_score (Credit Score): uma das variáveis mais importantes na previsão.

income_annum: rendimento anual do cliente.

loan_amount: valor solicitado.

no_of_dependents, loan_term e valor dos ativos: variáveis relacionadas ao risco e à capacidade de pagamento.

## Arquivo Necessário

Dataset: Loan_approval_data_2025.csv

Fonte: Kaggle – Realistic Loan Approval Dataset | US & Canada
Disponível em: https://www.kaggle.com/datasets/parthpatel2130/realistic-loan-approval-dataset-us-and-canada

## Objetivos do Projeto

> Explorar e limpar os dados

> Analisar correlações e padrões

> Treinar modelos de classificação

> Avaliar métricas como Acurácia, Precision, Recall e F1-Score

> Comparar o desempenho entre diferentes algoritmos
